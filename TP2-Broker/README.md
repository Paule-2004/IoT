Risques si MQTT n'est pas sécurisé ? Interception de données (MITM) et injection de commandes frauduleuses.
Différence TLS/mTLS ? Le TLS authentifie le serveur, le mTLS (Mutual TLS) authentifie le serveur ET le client via des certificats.
Pourquoi les ACL ? Pour limiter les privilèges : un capteur ne doit pouvoir écrire que sur son propre topic
